    GRender

    🛠️ A lightweight canvas library for 2D.

    快速上手 →

    # 起步

    安装：

    npm i grender
# or
yarn add grender

    上手

    import GRender, { Rect } from 'grender'

const grender = new GRender(document.querySelector('#app'))
const rect = new Rect({
  shape: {
    x: 0,
    y: 0,
    width: 40,
    height: 80
  },
  z: 2
})

grender.add(rect)
